# TXT 目录规则帮助

## 菜单区

- `添加`：新增目录规则，用于补充内置规则无法覆盖的标题格式。
- `本地导入`：从本地文件导入规则（支持 `txt` / `json`）。
- `网络导入`：输入链接导入在线规则，常用于同步或快速调试。
- `二维码导入`：扫描二维码导入规则。
- `导入默认规则`：恢复内置默认规则。不会删除用户自定义规则，但会覆盖同 ID 的默认规则项。

## 列表操作

- 单条规则支持：`置顶`、`置底`、`删除`。
- 批量操作支持：`启用所选`、`禁用所选`、`导出所选`、`删除`。
- 规则右侧显示 `已禁用` 时，表示该规则不会参与自动识别。

## 编辑页

- `保存`：保存当前规则。
- `复制规则`：将当前规则 JSON 复制到剪贴板。
- `粘贴规则`：从剪贴板读取 JSON 并覆盖当前编辑内容。
- `名称` 不能为空；`正则` 语法错误时无法保存。
